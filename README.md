### Описание проекта

DSSim построен как дискретно-событийный симулятор распределённой сети. Узлы и направленные каналы задаются графом смежности, а все действия системы представлены событиями: отправка, доставка, потеря сообщения, срабатывание таймера и отказ компонента.

События обрабатываются в порядке виртуального времени через приоритетную очередь. Это позволяет воспроизводимо моделировать работу сети без реальных задержек, потоков и сокетов.

Реализованы четыре алгоритма распространения сообщений:

* `unicast` — последовательная передача;
* `broadcast` — рассылка всем соседям;
* `multicast` — распространение по BFS-дереву;
* `gossip` — вероятностная рассылка случайным соседям.

Модель сети поддерживает задержку, джиттер, потери, дублирование и переупорядочивание сообщений, а также отказы узлов и отдельных направлений каналов.

Симуляция полностью рассчитывается до начала визуализации. Результат сохраняется как журнал кадров со состояниями узлов и каналов. PySide6-интерфейс не выполняет вычисления симуляции — он только воспроизводит готовый журнал, поддерживает паузу, перемотку и изменение скорости просмотра.


### Запуск проекта

Требуется Python 3.11 или новее и менеджер пакетов `uv`.

После распаковки архива необходимо открыть терминал в корневом каталоге проекта и установить зависимости:

```bash
uv sync
```

Запуск графического интерфейса PySide6:

```bash
uv run dssim-qt
```

При первом запуске `uv` автоматически создаст виртуальное окружение и установит PySide6, NetworkX и PyQtGraph.
