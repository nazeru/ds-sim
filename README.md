### Описание проекта

DSSim — дискретно-событийный симулятор распространения сообщений в распределённой сети. Пользователь выбирает топологию, алгоритм распространения и параметры сети: задержку, потери, дублирование сообщений и отказы узлов или каналов.

В проекте реализованы алгоритмы unicast, broadcast, multicast и gossip. Симулятор рассчитывает последовательность событий в виртуальном времени и формирует полный журнал работы сети.

После завершения расчёта журнал передаётся в PySide6-интерфейс. Интерфейс не выполняет вычисления симуляции, а только воспроизводит готовые события, отображает состояния узлов и каналов и показывает итоговые метрики.

### Запуск проекта

Требуется Python 3.11 или новее и менеджер пакетов `uv`.

После распаковки архива необходимо открыть терминал в корневом каталоге проекта и установить зависимости:

```bash
uv sync
```

Запуск графического интерфейса PySide6:

```bash
uv run dssim-qt
```

При первом запуске `uv` автоматически создаст виртуальное окружение и установит PySide6, NetworkX и PyQtGraph.
